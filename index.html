Philter Comparison

  • Designed a data pipeline and framework using Python for comparing the performance of UCSF’s clinical note de-identification software to an alternative algorithm in order to pinpoint notes that UCSF’s algorithm was not able to de-identify completely.
  • Devised a method of output comparison using numpy, scipy, and scikit-learn that flagged notes which caused the two algorithms to produce contrasting results over metrics such as the percentage of correctly identified PHI tokens and the cosine similarities.
  • Created a Python script for preprocessing output files into an XML file format mimicking gold standard PHI annotations.
